# @mtngtools/utils-core

Core utility functions shared across mtngTOOLS packages. Part of the **UTILS** layer (domain-agnostic; see [code organization](https://github.com/mtngtools/overview/blob/main/code-organization.md)).

## Install

```bash
pnpm add @mtngtools/utils-core
```

## Usage

```ts
import { identity } from '@mtngtools/utils-core';
```

## Development

```bash
pnpm run typecheck
pnpm run lint
pnpm run build
pnpm run test
```
